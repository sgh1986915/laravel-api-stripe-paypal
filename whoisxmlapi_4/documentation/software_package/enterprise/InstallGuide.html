Software Prerequiresite for installing Whois API:

  • Java Virtual Machine(JVM)
  • Tomcat application server(or other application server that supports servlet engine)
  • MYSQL(other db servers are not tested)

Installation of MYSQL databases:

1. Install MYSQL database server if it's not installed already. http://dev.mysql.com/downloads/mysql/, for a quick proof of concept you might want to set your root password to root, which matches the default password set in Whois API package.

2. Execute whoiscrawler.sql and whoisapi.sql in db_scripts against your database server, upon successful execution, you should see two new databases called whoiscrawler and whoisapi.

Setup Whois API configuration files:

1. copy all files from config/ into c:\whoiscrawler\ (or /home/whoisxmlapi/ on liunx)

Deploy whoisapi webapp:

1. Install Tomcat 6.0 if it's not installed already. http://tomcat.apache.org/download-60.cgi

2. Copy lib/tslib-win.jar into Tomcat's library(eg. C:\Program Files\Apache Software Foundation\Tomcat 6.0\lib, or /opt/tomcat/lib/)

3. Deploy whoisserver.war. This can be done by simply copying whoisserver.war into tomcat's webapps folder.

4. Test querying whois information using the default user root http://localhost:8080/whoisserver/WhoisService?domainName=test.com&username=root&password=root

Customization:

There are several configuration files within whois api.

1. whoisserver\WEB-INF\classes\general.properties, this file is the root property file that points to several other files:

a. whois.conf is the file that configures whois registries, registrars and other settings, in general you do not need to modify this file. It points to c:\whoiscrawler\whois.conf (or /home/whoisxmlapi/whois.conf) by default

b. proxy.conf is the file that contains a list of proxy servers that whois api uses. It points to c:\whoiscrawler\proxy.conf (or /home/whoisxmlapi/proxy.conf) by default.

c. httpproxy.conf is the file that contains a list of http proxy servers that whois api uses. It points to c:\whoiscrawler\httpproxy.conf (or /home/whoisxmlapi/httpproxy.conf) by default.

2. whoisserver\WEB-INF\lib\whoiscrawler.jar\hibernate_whoiscrawler.cfg, this file contains the database configuration such as database path pointing to whoiscrawler database. The default db url is jdbc:mysql://localhost:3306/whoiscrawler. The default username is root, the default password is root. You must set the correct database username and password here.

3. whoisserver\WEB-INF\classes\hibernate_whoisapi.cfg.xml, this file contains the database configuration such as database path pointing to whoisapi database. The default db url is jdbc:mysql://localhost:3306/whoisapi. The default username is root, the default password is root. You must set the correct database username and password here.
